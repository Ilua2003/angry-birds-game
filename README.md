# Angry Birds Clone

Angry Birds clone game built with JavaScript.

**Play now:** https://ilua2003.github.io/angry-birds-game/

## ⚠️ System Requirements
- **Desktop only** - optimized for PC/Mac browsers
- **WebGL support required**
- Modern browser recommended (Chrome, Firefox, Edge, Safari)
- Hardware acceleration enabled

## Description
A browser-based clone of the popular Angry Birds game, created for educational purposes to demonstrate JavaScript game development.

## Features
- Physics-based gameplay
- Multiple levels  
- Desktop-optimized
- Instant loading
